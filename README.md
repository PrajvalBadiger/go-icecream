# Go RESTful JSON API

This is a Restfull json API in go to add(POST), fetch(GET), remove(DELETE) and update(PUT) icecream falvours.

## Dependencies
Built using minimal external dependencies
1. github.com/gorilla/mux
2. github.com/lib/pq

## Build
```
make build
```

## Run
### 1. watch mode
```
make watch
```
### 2. run once
```
make run
```
